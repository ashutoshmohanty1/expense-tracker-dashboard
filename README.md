# Expense Tracker Dashboard

## Overview

Expense Tracker Dashboard is a React-based web application that helps users manage their daily expenses efficiently. Users can add, edit, delete, search, and filter expenses while tracking their spending through an interactive dashboard.

---

## Features

* Add new expenses
* Edit existing expenses
* Delete expenses
* Search expenses by name
* Filter expenses by category
* View total expenses
* View total transactions
* Persistent data storage using JSON Server
* Responsive dashboard UI

---

## Installation

### Clone Repository

### Install Dependencies

npm install

### Start JSON Server

npm run api

### Start React Application

npm run dev
